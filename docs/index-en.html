Processing Eventc back to ToC or Class ToC

IRI: https://www.morphosource.org/terms/morphosource#ProcessingEvent

An active event in which a media is processed to produce another media. Belongs to Imaging Event or Media as a member (as either of these can lead to a processing event), and has member Media (as the processing event leads to the production of Media).
has super-classes
Work c
is in domain of

Workc back to ToC or Class ToC

IRI: http://pcdm.org/works#Work

A work or intellectual entity, such as a book, film, dissertation, etc. Works have member Objects representing their physical structure (e.g., pages in a book), FileSets to maintain representations of the entire resource (e.g. a PDF of all the pages), and TopRanges representing their logical structure or structures (e.g., sections and chapters in a book).
Is defined by
http://pcdm.org/works#
has super-classes
object c
has sub-classes
Device c, Imaging Event c, Media c, Organization c, Physical Object c, Processing Event c
